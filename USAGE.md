## Data Structure

### SQLite Database

**Location:** `twitter_growth.db`

**Tables:**
- `growth` - Daily follower/following/tweets count
- `tweets` - Tweet performance metrics

**Query examples:**
```sql
-- Last 7 days growth
SELECT date, followers FROM growth ORDER BY date DESC LIMIT 7;

-- Best performing tweets
SELECT text, likes, retweets FROM tweets ORDER BY likes DESC LIMIT 5;

-- Daily follower change
SELECT 
  date,
  followers,
  followers - LAG(followers) OVER (ORDER BY date) as daily_change
FROM growth;
```

### JSON Export

**Location:** `data.json`

Generated by `collect_data.py` for frontend consumption.

## Troubleshooting

**No data showing:**
- Run `python3 collect_data.py` first
- Check `data.json` exists
- Check browser console for errors

**Authentication errors:**
- Make sure `~/.config/bird/.env` has AUTH_TOKEN and CT0
- Test: `bird whoami` should work

**Chart not rendering:**
- Check Chart.js CDN is loading
- Check `data.history` has data points
